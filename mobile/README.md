# Habits

## Technologies used:

- Ionic
- Vue 3

## Project Setup

```sh
npm install
```

### Compile and Hot-Reload for Development

```sh
ionic serve
```

### Add iOS and Android

```sh
npx cap add ios
npx cap add android
```

### Sync Files

```sh
npx cap sync
```

### Running iOS and Android

```sh
npx cap run ios
npx cap run android
```
